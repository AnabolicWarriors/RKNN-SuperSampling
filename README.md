# RKNN-SuperSampling

RK3399Pro 에서 NPU를 사용하고 최적화 하기 위한 프로젝트.



## 어떻게 빌드를 하나요?

### PreRequirement 

> 1. wget<br>
> 2. tar<br>
> 3. Docker<br>
> 4. 네트워크 가능한 환경<br>

### MacOS / Ubuntu 기준

```sh
chmod +x ./bootstrap.sh
./bootstrap
```

### Windows 기준

```sh
# 아직 미 지원기능, 누가 윈도우 씀 ㅎ?
chmod +x ./bootstrap.bat
./bootstrap
```
